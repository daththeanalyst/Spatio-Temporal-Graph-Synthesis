# Competency Tracker: Geospatial Retail Analyst

This document maps the specific skills you will acquire across the 3 notebooks of this project.

| Phase | Skill | Industry Relevance |
| :--- | :--- | :--- |
| **01: Ingest** | **CRS Management** | Understanding how to project the Earth onto a flat map—essential for accurate logistics and site scoring. |
| **01: Ingest** | **Multi-Format ETL** | Handling Rasters (pixels), Vectors (points/polygons), and CSVs (tables) in a single pipeline. |
| **02: Indexing** | **H3 Hexagonal Logic** | Mastering Uber's global grid system—the industry standard for big data spatial analytics (Uber, Grab, DoorDash). |
| **02: Indexing** | **Zonal Statistics** | Learning how to "extract" tabular data from visual maps (rasters). |
| **03: Analytics** | **Graph Topology** | Representing cities as networks of nodes and edges, allowing for advanced connectivity analysis. |
| **03: Analytics** | **Structural Hole Theory** | Identifying market gaps where demand exists but supply is missing—highly valued in retail strategy roles. |
| **03: Analytics** | **3d Geospatial DataViz** | Creating high-end, interactive stakeholders reports using Pydeck or Kepler.gl. |

## Mentoring Path
By the end of this series, you will have moved from a "Spreadsheet Analyst" to a "Spatial Data Scientist," capable of building decision-support systems for enterprise retail expansion.
